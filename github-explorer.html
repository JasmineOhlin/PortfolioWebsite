Github Explorer

Team size: Solo

Duration: 3 weeks

Softwares: JetBrains Rider and C#

Assignment: Create a small GitHub Repository browser by accessing GitHub's public REST API to receive information.

About
By using GitHub REST APIs I created a small browser in the IDE that could:

● Commit and display issues in a repository.

● Show a user's profile, repositories and organizations.

● Visit a user's repositories and organizations from their profile.

● List members of an organization.
