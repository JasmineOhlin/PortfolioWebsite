Web Crawler

Team size: Solo project

Duration: 3 weeks

Softwares: Jetbrains Rider, C# and Regex

Assignment: Create a small browser that reads and retrieves information from a websites HTML and prints it in the IDE console using HTTP methods and a TCP client. This project was one of several TCP/UDP projects made during 3 weeks.

About
By using TCP and UDP this IDE application is used to pick out information from a websites HTML and display it neatly in the console. Added functionality was to navigate between links, visit them and retrieve information.

Apart from the demands for this assignment I also explored Regex to more efficently pick out information from the HTML document.
